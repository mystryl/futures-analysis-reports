期货回测系统 API

欢迎使用期货回测系统

这是一个基于 Flask 的期货回测系统，提供丰富的数据分析和可视化功能。

可用的 API 端点

品种搜索 API

端点: /api/symbols

方法: GET

参数: q (可选) - 搜索关键词

示例:

  • /api/symbols - 获取所有品种
  • /api/symbols?q=rb - 搜索螺纹钢相关品种

历史数据 API

端点: /api/history

方法: GET

参数:

  • symbol - 期货品种代码（必需）
  • period - 周期（可选，默认为 daily）
  • start_date - 开始日期（可选）
  • end_date - 结束日期（可选）

回测 API

端点: /api/backtest

方法: POST

参数: 策略配置参数
